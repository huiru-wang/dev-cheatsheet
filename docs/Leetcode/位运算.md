
# 常用位运算

## 判断奇偶

偶数：`x & 1 == 0`

奇数：`x & 1 == 1`

## 取某数的第k位

`x >> k & 1`

## 取相反数

`x >> k & 1`

## 使用异或交换两数

```java
// 交换a，b的值，不可用于浮点数
a ^= b;
b ^= a;
a ^= b;
```

## &代替取余

`X % n == X & (n-1)`，前提：n为2的整数次幂；
>参考HashMap

假设length是8（2^3）

` X / 8` = ` X / 2^3 ` = ` X >> 3 `，同时右移挤掉的低三位，就是余数；

` X & (n-1) `相当于取 X 值的(n-1)个低位bit；相当于直接取余数；


## 将n的最后一个1变更为0

`n & (n−1)`

n-1：把n的最后一个1变为0，剩余低位变为1；
